# Privacy Policy

*Last updated: January 1, 2025*

## Introduction

Snap4Knack ("we," "our," or "us") is operated by Fine Mountain LLC. This Privacy Policy explains how we collect, use, and protect information when you use our visual feedback platform.

## Information We Collect

### Account Information
When you create an account, we collect your name, email address, and password (stored as a secure hash via Firebase Authentication).

### Usage Data
We collect data about how you use the platform, including pages visited, features used, and session duration.

### Feedback Submissions (Snaps)
When a Knack application user submits feedback, we collect:
- Screenshot or screen recording of the page
- Browser and device information
- Page URL at time of capture
- Any annotations or text entered by the user
- Optional console error logs

### Knack Integration Data
We store Knack application IDs and API keys (encrypted in Google Cloud Secret Manager). We do not store Knack user passwords.

## How We Use Your Information

- **To provide the service**: Process and display feedback submissions
- **To send notifications**: Alert team members of new submissions per their preferences
- **To improve the platform**: Analyse usage patterns to enhance features
- **To communicate**: Send service-related emails

## Data Storage and Security

All data is stored in Google Firebase (Firestore, Cloud Storage) in the United States. We use:
- Firebase Authentication for access control
- Firestore security rules for data isolation between tenants
- Google Cloud Secret Manager for API key storage
- HTTPS encryption for all data in transit

## Data Sharing

We do not sell, trade, or rent your personal information. We may share data with:
- **Firebase / Google Cloud**: Infrastructure providers
- **SendGrid**: Email delivery service
- **Legal requirements**: If required by law

## Data Retention

Feedback submissions are retained until deleted by the tenant. Account data is retained while your account is active. You may request deletion by contacting us.

## Your Rights

You have the right to:
- Access the personal data we hold about you
- Request correction of inaccurate data
- Request deletion of your account and associated data
- Opt out of non-essential communications
